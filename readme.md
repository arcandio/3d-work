## PBR

### PBR Workflow

* Concept Art / Studies (Clip / Krita)
* Lowpoly / sketch / basemesh (Blender)
* Highpoly (Blender / Zbrush)
* Texture Paint (Blender / Zbrush)
* Refine basemesh (blender)
* Bake (blender / Xnormal, maybe Zbrush)
* Preview/view (Sketchfab)
